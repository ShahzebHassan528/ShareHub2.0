## 📋 FORMATTING REQUIREMENTS (Page 10 - LGU Handbook)

### Font & Style
1. **Body Text**: Times New Roman, 12pt
2. **Chapter Heading (Heading1)**: Times New Roman, Bold, 20pt, Right Aligned
3. **Topic Heading (Heading2)**: Arial, Bold, Italic, 14pt, Left Aligned
4. **Sub Heading (Heading3)**: Arial, Bold, 13pt, Left Aligned

### Alignment & Spacing
- **Paragraph**: Justify alignment
- **Paragraph Spacing**: 6pt after, 12pt before
- **Line Spacing**: 1.5 lines
- **Topics**: Left alignment

### Page Numbering
- **Before Chapter 1**: Roman numerals (i, ii, iii, iv, v) - Bottom Right
- **From Chapter 1 onwards**: Arabic numerals (1, 2, 3...) - Bottom Right

### Captions
- **Tables**: Caption on TOP
- **Figures**: Caption on BOTTOM

### References Format
**Journals/Conferences:**
```
Author Name (Surname, Initial), (Year), "Title", Journal/Conference Name, vol. X, no. Y, pp. Z-Z.
```

**Books:**
```
Author Name (Surname, Initial), (Year), Title, Publisher, Place, pp. X-Y.
```

**Internet:**
```
Author Name (if known), "Title", URL
```

## 📄 REPORT STRUCTURE (Annex C)

### Cover Page
- Project Title
- Student Names (Program-Roll#-Section)
- Session
- Supervisor Name
- Department & University

### Front Matter (Roman Numerals)
1. Title Page
2. Copyrights Page
3. Declaration Page
4. Acknowledgements
5. Dedication
6. Table of Contents
7. List of Tables
8. List of Figures
9. List of Abbreviations
10. Abstract (1 page max)

### Main Content (Arabic Numerals)
1. Chapter 1: Introduction
2. Chapter 2: Problem Definition
3. Chapter 3: Software Requirement Specification
4. Chapter 4: Methodology
5. Chapter 5: System Architecture/Initial Design
6. Chapter 6: Implementation and Testing
7. Chapter 7: Results and Discussion
8. Chapter 8: Conclusion and Future Work
9. References
